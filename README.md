## 🎮 Sobre el Proyecto

**Moustache Battle Royale** es una aplicación web desarrollada en Laravel 10 que permite gestionar "guerras" entre diferentes equipos. El sistema automatiza los enfrentamientos y lleva un registro detallado de las bajas, estadísticas y el historial de campeones.

### Características principales:
- **Gestión de Guerras:** Creación, inicio y finalización de eventos de combate.
- **Sistema de Equipos:** Organización por presidentes y reclutamiento de jugadores.
- **Lógica de Asesinatos:** Registro en tiempo real de quién mata a quién.
- **Estadísticas Avanzadas:** Ranking de asesinos, mejores equipos y progreso de la guerra actual.
- **Interfaz Arcade:** Diseño oscuro con fondo y fuentes pixel art y físicas interactivas.

## 🛠️ Stack Tecnológico

* **Backend:** Laravel 10 (PHP 8.1+)
* **Frontend:** Blade, Bootstrap 5.3, Google Fonts (Press Start 2P), CSS.
* **Interactividad:** JavaScript Vanilla (Motor de físicas y colisiones).
* **Base de Datos:** SQLite (Desarrollo), MySQL (producción).
